# LibreTranslate App for Akeneo PIM

Translates product content between locales within Akeneo PIM

## Requirements
- Docker & Docker Compose
- WSL (Windows Subsystem for Linux)
- Python 3.12+ 

### Local Services (Docker)
- **Akeneo PIM Community Edition**
- **LibreTranslate** 
> Both are started via 'docker-compose.yml'

## Notes
If your local Akeneo clone uses `docker-compose` in its Makefile, replace it with `docker compose` before running `make`.
